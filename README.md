# Yelp Dataset SQL Analysis 🍽️

A SQL analysis project exploring the Yelp dataset to uncover business trends, user behavior, and review patterns. Completed as part of a Coursera SQL course.

## Dataset
- 11 tables including Business, Review, User, Category, Checkin
- 10,000 records per table

## Key Analysis
- Profiled all 11 tables for total and distinct records
- Identified cities with most reviews (Las Vegas leads with 3,873)
- Analyzed correlation between reviews and fans
- Found "love" appears 1,780 times vs "hate" 232 times in reviews
- Compared open vs closed businesses by category and state
- Predicted business viability using stars, reviews, and customer feedback

## SQL Concepts Used
- JOINs (INNER, LEFT JOIN)
- GROUP BY and ORDER BY
- CASE statements
- Aggregate functions (COUNT, MIN, MAX, AVG)
- LIKE for text pattern matching
- Subqueries

## Tools Used
- **SQL (SQLite)** — data profiling and analysis
